Кто людям помогает –
Лишь тратит время зря, ха-ха,
Хорошими делами
Прославиться нельзя.
Поэтому я всем и каждому советую
Всё делать точно так,
Как делает старуха
По кличке Шапокляк.
Как делает старуха
По кличке Шапокляк!
